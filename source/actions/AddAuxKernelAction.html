AddAuxKernelAction

This action serves as a shortcut to add an AuxKernel nested within an AuxVariable block in the input file syntax.

The syntax is


[AuxVariables]
  [foo]
    order = SECOND
    [AuxKernel]
      type = bar
    []
  []
[]

See AuxKernel for more information about AuxKernels.
