Material Vector Force Action System

Set up volumetric body force kernels

Description

The MaterialVectorBodyForce Action is used to create a set of body force kernels for each displacement variable of the mechanical problem.

See the description, example use, and parameters on the MaterialVectorBodyForce kernel page.
