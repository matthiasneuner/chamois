ThermalFractureIntegral

Calculates summation of the derivative of the eigenstrains with respect to temperature.

Description

This model computes the summation of the derivative of the eigenstrains with respect to temperature. This quantity is used in the computation of fracture integrals in the presence of temperature. This object is not typically defined by a user, but is set up automatically using the DomainIntegralAction when temperature is a coupled variable.
